# Angela y el Reino Arcoiris de las Palabras

App educativa para Angela, 9 anos, recien terminada 3 de Primaria y preparando el paso a 4.

Objetivo: aprender Lengua Castellana de forma diaria, corta y divertida, con ejercicios adaptativos, lectura, ortografia, gramatica, vocabulario y expresion escrita.

## App publicada

- GitHub Pages: https://evaristosaa.github.io/lengua-angela/

## Enfoque

- Sesiones cortas de 10-15 minutos.
- Mucho juego, poca ficha escolar disfrazada.
- Progreso visible para Angela y resumen simple para padres.
- Contenido alineado con 3-4 de Primaria, segundo ciclo LOMLOE.
- Funciona bien en movil/tablet y tambien en navegador.
- Narrativa principal: Angela cumple misiones de Lengua y va formando un unicornio de arcoiris.
- Estilo visual: Angela tipo comic en distintas poses, celebraciones y acciones dentro de la app.
- Requisito tecnico no negociable: publicable en GitHub Pages, con rutas y assets funcionando desde subruta.
