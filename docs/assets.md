# Assets

Los assets finales viven en `public/assets/`.

## Angela

Usar los nombres acordados en `docs/assets-angela.md`. Si una imagen aun no esta disponible, la app debe mostrar un placeholder con el estado correspondiente.

## Unicornio

Los elementos del unicornio deben guardarse en `public/assets/unicornio/` con nombres descriptivos:

- `crin-arcoiris.png`
- `alas-brillantes.png`
- `corona-dorada.png`
- `estela-magica.png`

Todos los assets deben funcionar desde una subruta de GitHub Pages.
